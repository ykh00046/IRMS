# IRMS 저울 에이전트 (A&D GX-10202M)

계량하는 **현장 PC**에서 실행하는 작은 프로그램입니다. 저울(RS-232C/USB-시리얼)을
읽어 `http://127.0.0.1:8787` 로 무게를 내어주고, 배합 화면이 그 값으로 실제량을
자동 입력합니다. 서버에 설치하는 게 아니라 **저울이 연결된 PC**에 설치합니다.

## 연결

- GX-10202M 뒷면 RS-232C(D-Sub 9핀) ↔ PC. PC에 시리얼 포트가 없으면
  USB-시리얼 컨버터 사용(장치관리자에서 COM 번호 확인).
- 저울 통신 설정을 바꾼 적 없으면 그대로 동작(공장 기본 2400bps/7bit/EVEN/1stop).
  바꿨다면 `%APPDATA%\IRMS-Scale\config.json` 의 baudrate 등을 저울 설정(bASFnc)과
  맞추세요. `port` 를 null 로 두면 COM 포트 자동 탐지.

## 실행

```
pip install -r requirements.txt   (최초 1회)
run_scale_agent.bat               (창을 켜 둔다 — 시작프로그램 등록 가능)
```

창에 `저울 연결됨: COM3` 이 뜨면 준비 완료. 배합 화면을 새로고침하면
실제량 칸 옆에 [저울] 버튼이 나타납니다.

## 사용 (배합 화면)

1. 자재를 저울에 올리고 표시가 안정될 때까지 대기
2. 해당 자재 행의 **[저울]** 버튼 클릭 → 실제량이 자동 입력되고 다음 행으로 이동
3. 허용 편차는 자재별 ±0.05 g — 초과하면 저장이 차단됩니다(합계 편차는 제한 없음)

## 문제 해결

- `저울을 찾지 못했습니다`: 케이블·전원·COM 번호 확인. 다른 프로그램(WinCT 등)이
  포트를 점유 중이면 종료.
- 값이 `stable: false`(측정 중)만 나오면: 저울이 흔들리거나 바람막이 열림.
- 브라우저에서 [저울] 버튼이 안 보이면: 이 에이전트 창이 떠 있는지, 그리고
  `http://127.0.0.1:8787/health` 가 브라우저에서 열리는지 확인.
