# 점도 분석 (viscosity-analysis) 기획서

## 목표
합성 점도(PB/SBCT/SCRA)를 LOT별로 등록하고, 제품별 추세와 이상 발생을 한
화면에서 관리한다. 기존 엑셀(`합성 점도.xlsx`) 운영을 시스템으로 대체.

## 사용자 요구 (의사결정 확정)
- 이상 판정: **관리 상/하한 + 통계(σ) 결합**
- 기존 1~6월 데이터: **전량 임포트**
- 제품군: **추가 가능**하게(마스터로 관리)
- 권한: **등록=작업자(operator), 설정=관리자(manager)**

## 범위
1. DB: viscosity_products / viscosity_readings + 제품 시드
2. 분석 서비스: 통계·관리한계·추세·이상 판정 + LOT 날짜 파서
3. API: 조회/등록(operator), 제품 설정/삭제/CSV(manager)
4. 화면 `/viscosity`: 등록 폼 · 추세 차트(관리한계) · 이상/추세 표기 · 이력
5. 엑셀 임포트 스크립트(멱등)
6. 단위/라우트 테스트

## 비범위
- 저울 자동 연계(현 계량과 동일하게 수동 입력)
- 운영 대시보드(/dashboard) 통합 카드는 차기(영문/한글 정책 충돌 회피)

## 수용 기준
- 제품별로 다른 정상 대역에서 이상이 올바르게 판정된다.
- 작업자는 등록 가능, 관리자 설정/삭제/내보내기는 작업자에게 403.
- 엑셀 252건이 멱등하게 적재된다(중복 LOT 제외).
- 전체 pytest 회귀 없음.
